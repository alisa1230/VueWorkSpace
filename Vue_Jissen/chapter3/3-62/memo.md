# 配列要素の追加と削除
* vueでは配列の要素を変更させた時検出することができない。<br>
  なので特定の関数を使って要素を変更する必要がある<br>
  | メソッド                                  | 概要                                                 |
  | :---------------------------------------- | :--------------------------------------------------- |
  | push                                      | 末尾に追加                                           |
  | pop()                                     | 末尾の要素を削除                                     |
  | shift()                                   | 先頭の要素を削除                                     |
  | unshift(elem,...)                         | 先頭に要素を追加                                     |
  | splice(star[,count[,elme1[,elem2 ,...]]]) | start番目からcount個の要素をelem1,elem2...要素で置換 |
  | sort()                                    | 要素を順番に並び替え                                 |
  | reverse()                                 | 要素の並び順を反転                                   |
<br>
